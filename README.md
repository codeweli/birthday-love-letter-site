# Birthday Love Letter Website

## Overview
This project is a cinematic, scroll-driven 3D happy birthday website designed as a digital love letter. It combines modern WebGL-style 3D elements, smooth parallax animations, and immersive storytelling to create a romantic interactive experience.

## Features
- Interactive 3D birthday scene
- Smooth parallax scrolling effects
- Pop-up letter modal for personalized messages
- Responsive design for various devices

## Project Structure
```
birthday-love-letter-site
├── src
│   ├── app.ts                # Entry point of the application
│   ├── components
│   │   └── letter-modal.ts   # Manages the interactive pop-up letter
│   ├── scenes
│   │   └── birthday-scene.ts # Creates the 3D birthday scene
│   ├── styles
│   │   └── main.css          # CSS styles for the website
│   └── types
│       └── index.ts          # Type definitions for TypeScript
├── public
│   └── assets                # Static assets (images, 3D models, etc.)
├── package.json              # npm configuration file
├── tsconfig.json             # TypeScript configuration file
└── README.md                 # Project documentation
```

## Setup Instructions
1. Clone the repository:
   ```
   git clone https://github.com/yourusername/birthday-love-letter-site.git
   ```
2. Navigate to the project directory:
   ```
   cd birthday-love-letter-site
   ```
3. Install the dependencies:
   ```
   npm install
   ```
4. Start the development server:
   ```
   npm start
   ```
